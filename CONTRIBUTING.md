# Contributing to AI RAG System

Thank you for considering contribution! This project welcomes all contributions.

## Development Setup

```bash
git clone https://github.com/walidsobhie-code/ai-rag-system.git
cd ai-rag-system
pip install -r requirements.txt
pytest
```

## Submit PRs

1. Fork the repo
2. Create feature branch
3. Commit changes
4. Push and open PR
